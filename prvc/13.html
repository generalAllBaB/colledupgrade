    Условные операторы и оператор выбора



      `if-else` контролирует выполнение кода на основании условия. Если условие истинно (1), выполняется действие. Если ложно (0), идёт дальше. Пример с одним действием:


      if (условие)
        действие;


      Если нужно несколько действий, их оборачивают в `{}`. Важно не ставить `;` сразу после условия, иначе блок действий всегда выполняется. Пример вложенного `if`:


      if (условие) {
        if (другое условие)
          действие;
        ещё действие; }


      Выражения вне скобок выполняются независимо от условий.


      Условные операторы `if` и `if-else` управляют потоком выполнения кода. Важно правильно использовать фигурные скобки для определения блоков кода. В `if` без скобок к условию относится только первое следующее действие. Вложенные `if` и `else` соединяются так, что каждый `else` относится к ближайшему предшествующему `if`. Чтобы избежать путаницы, используйте фигурные скобки и делайте отступы. Это упрощает чтение кода и снижает риск ошибок. Конструкция `if-else if` позволяет прекратить проверку дополнительных условий, как только одно из них выполнено.


      В программировании на C при решении квадратного уравнения используйте структуру условного оператора `if-else` для проверки дискриминанта (`D`): если `D < 0`, корней нет; если `D == 0`, два совпадающих корня; если `D > 0`, два различных корня. Выражение в `if` истинно, когда не равно нулю. Логические операции: `&&` (и), `||` (или), `!` (не). Приоритет операции `==` выше, чем у `=`. Например, используйте `if (a >= 20 && a <= 23)` для проверки попадания в диапазон. Не путайте операции сравнения (`==`) и присваивания (`=`).


      Тернарный оператор `?:` в языке C позволяет выполнять одно из двух выражений в зависимости от истинности условия:


      int max, x = 5, y = 4;


      max = (x > y) ? x : y;


      `switch` используется для множественного выбора на основе значения переменной:


      switch (переменная) {
        case значение1: действие1; break;
        case значение2: действие2; break;
        // ...
        default: действиеПоУмолчанию;
      }


      `break` в конце каждого `case` предотвращает "проваливание" в следующий случай. Без `break` выполнение продолжится с последующих `case`.


      Вопросы:


      1. Как работает условный оператор if-else?


        Условный оператор if-else работает следующим образом: при выполнении условия внутри if выполняется код внутри блока if, иначе, если условие не выполнено, выполняется код в блоке else.


      2. Какие конструкции можно составить условным оператором if-else?


        С условным оператором if-else можно сформировать конструкции, такие как if-else if (продолжение проверок при несовпадении предыдущих условий) и вложенные условия.


      3. Какие операции отношения и логические операции используются в языке Си?


        Операции отношения (<, >, <=, >=, ==, !=) используются для сравнения переменных. Логические операции (&& - логическое И, || - логическое ИЛИ, ! - логическое НЕ) позволяют комбинировать условия.


      4. Основные ошибки и недостатки при работе с логическими операциями и операциями отношений


        Основные ошибки при работе с логическими операциями и операциями отношений: неверное использование скобок, ошибки при сравнении переменных разных типов, неправильное понимание приоритета операций.


      5. Суть операция «условие» ? :


        Операция «условие» ? : является тернарным условным оператором. Он позволяет выбирать между двумя значениями на основе условия.


      6. Назначение оператора выбора switch


        Оператор выбора switch используется для выбора между множеством возможных действий в зависимости от значения выражения.


      7. Приведите примеры использования оператора switch с (и без) оператора break


        - Пример с оператором break:


          switch (выражение) {
            case значение1:
            // действия
            break;
            case значение2:
            // действия
            break;
            default:
            // действия по умолчанию
          }


          - Пример без оператора break:


          switch (выражение) {
            case значение1:
            case значение2:
            // действия
            break;
            default:
            // действия по умолчанию
          }
