    Консольный ввод-вывод. Форматированный вывод. Форматированный ввод. Другие функции консольного ввода-вывода



      Функция scanf() используется для форматированного ввода данных с клавиатуры. Для обращения к элементу массива ввода нужно указать имя массива и индекс в квадратных скобках. Спецификатор формата определяет правила интерпретации входных данных. В вызове scanf() форматная строка определяет взаимодействие с аргументами. Ошибки scanf(): неправильное разделение спецификаторов, отсутствие амперсанда перед аргументом, неверное использование спецификаторов формата. Другие функции консольного ввода-вывода: getche() (читает и отображает символ), getch() (читает без отображения), getchar() (буферный ввод), putchar() (вывод символа), gets() и puts() (ввод и вывод строк).


      Вопросы:


      1. Какая функция отвечает за форматированный вывод и что делает литера f?


        Функция отвечающая за форматированный вывод - printf(). Литера 'f' после обратного слэша '\' используется для экранирования специальных символов.


      2. Что может включать форматная строка?


        Форматная строка может включать спецификаторы формата для переменных (например, %d для int, %f для float).


      3. Что делают управляющие символы?


        Управляющие символы в форматной строке обращаются к особым действиям, таким как перенос строки (\n), табуляция (\t) и другие.


      4. Какие элементы спецификации преобразования являются обязательными?


        Обязательными элементами спецификации преобразования являются знак %, а также спецификатор формата, который указывает на тип переменной.


      5. На что указывает спецификатор формата?


        Спецификатор формата указывает на тип переменной, которую нужно вывести или прочитать (например, %d для int, %f для float).


      6. Что такое ширина поля?


        Ширина поля определяет количество символов, отведенных для переменной в выводе.


      7. Дайте определение понятию «точность»


        Точность - это параметр, указывающий количество цифр после запятой для чисел с плавающей точкой.


      8. Что выводится при отсутствии спецификатора «точность» для вещественных чисел?


        При отсутствии спецификатора "точность" для вещественных чисел обычно выводится полное число без дробной части.


      9. Что делает функция scanf?


        Функция scanf используется для форматированного ввода данных с консоли в программу.


      10. Какие основные ошибки при работе с функцией scanf?


        - Неправильное использование спецификаторов формата.


        - Необработанные символы new-line (\n) или другие управляющие символы могут привести к некорректному вводу.


      11. За что отвечают функции getche, getchar, putchar, gets, puts?


        - getche(): читает одиночный символ с экрана с отображением вводимого символа.


        - getchar(): читает одиночный символ с экрана без отображения вводимого символа.


        - putchar(): выводит одиночный символ на экран.


        - gets(): читает строку с экрана.


        - puts(): выводит строку на экран.
