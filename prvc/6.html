    Введение. Основы программирования



    Язык программирования C был создан в 1972 году Деннисом Ритчи из Bell Laboratories. Разработан как замена ассемблера для ОС, C обеспечивает переносимость кода благодаря аппаратной независимости. Unix - первая ОС на C. Большинство современных ОС используют C/C++. API Windows базируется на C. С развитием вычислительных задач C эволюционировал, что привело к созданию Си++ Бьерном Страуструпом в 1980-х для объектно-ориентированного программирования. C поддерживает модульность, блочность, раздельную компиляцию и низкоуровневый доступ к памяти. Гибкость и универсальность C объясняют его популярность, высокую скорость и возможность работы с памятью. В Си любая программа состоит из функций, начиная с main - единственной и обязательной.

    Вопросы:


    1. В чём преимущества языка Си?


      - Эффективность: Си - низкоуровневый язык, близкий к аппаратной части, что обеспечивает хорошую скорость выполнения программ.


      - Портируемость: Код на C можно легко переносить между различными платформами.


      - Гибкость: Возможность написания прямого управления памятью и доступа к аппаратуре, что дает большой контроль над программой.


      - Богатая стандартная библиотека: В C множество функций доступно стандартно, что делает его мощным для различных программ.


    2. Поясните суть функции с именем main?


      - Функция main() является точкой входа для исполнения программы на C. Все программы на C должны содержать функцию main(), которая запускается первой.


      - Она возвращает целочисленное значение (int), обычно 0, что обозначает успешное завершение программы.


    3. Что такое переменные и константы?


      - Переменные: Это именованные области памяти, которые могут содержать изменяемые данные.


      - Константы: Это значения, которые не могут быть изменены в процессе выполнения программы.


    4. Что делают операции и выражения?


      - Операции выполняют действия над операндами и могут быть арифметическими, логическими, присваивания и др.


      - Выражения: Сочетание переменных, констант и операций, которые могут быть вычислены для получения значения.


    5. Приведите примеры математических функций стандартной библиотеки


      - sqrt(x) - вычисляет квадратный корень числа x.


      - sin(x), cos(x), tan(x) - вычисляют синус, косинус, тангенс угла x (в радианах).


      - exp(x) - вычисляет экспоненту числа x.


    6. Зачем перед именем каждой функции следует помешать тип возвращаемого значения?


      Это указывает на тип данных, который должен вернуться из функции. Обеспечивает строгое соблюдение типов данных и позволяет компилятору проверить правильность использования функции.
