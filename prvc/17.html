Лекция №17-18



    Операторы перехода



      Оператор continue используется для пропуска оставшейся части текущей итерации цикла и сразу перехода к следующей итерации. Применяется в циклах while, do-while и for.


      Оператор break прекращает выполнение текущего цикла или switch и передает управление следующему оператору, следующему за прерванным циклом или switch.


      Решение задачи с оператором break:


      int k,j,a=6;
      for(k=0; k<3; k++) {
        if(k>a) break;
        j=a;
        while(j >0) {
          a -= j;
          j -= 2;
        }
      }


      После выполнения кода переменная `a` равна 3.


      Оператор `goto` выполняет безусловный переход на указанную метку внутри функции. Он позволяет пропустить часть кода и продолжить выполнение с места метки. Однако его использование не рекомендуется, так как это может привести к плохо структурированному и запутанному коду. Вместо `goto` лучше использовать структурированные конструкции управления, такие как циклы и условные операторы.


      Оператор `return` используется для завершения функции: без возврата значения (`return;`) или с возвратом результата выражения (`return выражение;`). В функции `main` возвращает значение операционной системе.


      Вопросы:


      1. Как работает оператор continue?


        - continue используется в цикле для пропуска оставшейся части тела цикла и перехода к следующей итерации.


        - При выполнении continue управление передается обратно к проверке условия цикла, пропуская оставшуюся часть текущей итерации цикла.


      2. Как найти конец тела цикла, если оно не заключено в фигурные скобки {}?


        - Если тело цикла не заключено в фигурные скобки, то конец тела цикла можно определить по следующему оператору цикла или другому оператору, который находится не сразу после тела цикла.


      3. Какие применения имеет оператор break?


        - break прекращает выполнение цикла или оператора switch.


        - Используется для выхода из цикла или switch, когда достигнуто определенное условие.


      4. Как работает оператор goto (безусловный переход)?


        - goto позволяет осуществить безусловный переход на метку в коде.


        - Использование goto не рекомендуется из-за его способности привести к сложному и труднопредсказуемому поведению программы.


      5. К чему приводит частое использование произвольных переходов?


        - Частое использование goto и других произвольных переходов может привести к усложнению программы.


        - Это создает путаницу и делает код менее читаемым и поддерживаемым.


      6. Какие варианты использования имеет оператор return?


        - return используется для возврата значения из функции.


        - Может использоваться для завершения выполнения функции и передачи значения обратно в вызывающую программу.


        - Также можно использовать return для преждевременного выхода из функции в середине ее выполнения.
