Лекция №15-16



    Циклы



      Цикл `while` в C выполняет блок кода, пока условие истинно:


      while (условие) {
        тело цикла;
      }


      Цикл `do-while` исполняет тело цикла хотя бы один раз, затем проверяет условие и, если оно истинно, повторяет цикл:


      do {
        // Тело цикла
      } while (условие);


      Оператор цикла `for` — это цикл с параметрами. Его форма:


      for (инициализация; условие; изменение) {
        тело цикла;
      }


      Вопросы:


      1. Как работает оператор цикла while (цикл с предусловием)?


        Оператор цикла while работает следующим образом: выполняет блок кода, пока условие истинно. Условие проверяется перед каждой итерацией.


      2. Что такое тело цикла. Как оно обозначается?


        Тело цикла - это блок кода, который выполняется в каждой итерации цикла. Оно обозначается фигурными скобками {}.


      3. Как работает оператор цикла do-while (цикл с постусловием)?


        Оператор цикла do-while работает подобно циклу while, но проверка условия выполняется после выполнения тела цикла. Таким образом, тело цикла выполняется хотя бы один раз.


      4. В чём особенности оператора цикла for (цикл с параметрами)?


        Особенности оператора цикла for заключаются в том, что он имеет удобный синтаксис, включая инициализацию счётчика, условие продолжения и изменение счётчика в одной строке.


      5. Какова последовательность выполнения цикла for?


        Последовательность выполнения цикла for: инициализация -> проверка условия -> выполнение тела цикла -> изменение параметров -> проверка условия и так далее.


      6. Как работают разделы цикла for (инициализация, условие, изменение)?


        - Инициализация (выполняется один раз в начале цикла).


        - Условие (проверяется перед каждой итерацией).


        - Изменение (выполняется в конце каждой итерации).


      7. Что такое вложенные циклы и когда они используются?


        Вложенные циклы - это циклы, находящиеся внутри других циклов. Они используются, например, для обработки двумерных структур данных, многократного применения определенной логики и т. д.
